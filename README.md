# CoinFlip

This simple C# console application allows you to play a coin flipping game where you guess whether the result will be heads or tails.

## Example

```
Heads or Tails?
Type H or T:
H
It was heads!
You won!
Do you want to try again?
Type Y or any key to quit:
Y
Heads or Tails?
Type H or T:
T
It was tails!
You won!
Do you want to try again?
Type Y or any key to quit:
n
```
